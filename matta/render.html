matta.render.build_tree_from_path(path='.', max_depth=3, current_depth=0, ignore_dirs=None)¶

Build a tree structure dictionary from filesystem paths.

Recursively traverses directories up to a maximum depth and creates a nested dictionary representation. Common development directories are ignored by default.

Parameters:

  • path (str) – Directory path to scan. Defaults to current directory.

  • max_depth (int) – Maximum depth to traverse. Defaults to 3.

  • current_depth (int) – Current recursion depth. Internal use only.

  • ignore_dirs (set) – Directory names to ignore. Defaults to {‘.git’, ‘__pycache__’, ‘.venv’, ‘node_modules’, ‘.pytest_cache’}.

Returns:

Nested dictionary where keys are folder names, values are subdirectories,

and “_files” key contains list of files in that directory.

Return type:

dict

matta.render.render_csv_table(csv_file: str, format: str = 'markdown') → tuple[str, int]¶

Render CSV test variations as a formatted table.

Parses semicolon-delimited CSV with test case parameters and variations.

CSV Convention:

  • Row 0 (header): col 0 = label, col 1 = ignored, col 2+ = test case names

  • Row 1+: col 0 = param name, col 1 = short label, col 2+ = values

Parameters:

  • csv_file (str) – Path to the CSV file.

  • format (str) – Output format, either “markdown” or “plaintext”. Defaults to “markdown”.

Returns:

(table_string, num_test_cases) where table_string is the formatted

table and num_test_cases is the count of test case columns.

Return type:

tuple
