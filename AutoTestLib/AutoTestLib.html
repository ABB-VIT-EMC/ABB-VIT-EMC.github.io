static should_be_within_percentage(val, val_expected, tol_percent=5)¶

Check if value(s) are within percentage tolerance.

Parameters:

  • val – Single value or list of values

  • val_expected – Single expected value

  • tol_percent – Single tolerance percentage or list of tolerances

Returns:

AssertionError if any value is outside the tolerance

Note

  • If val is a list and tol_percent is a single value, the same tolerance will be applied to all values.

  • If both val and tol_percent are lists, they must be of the same length.

  • If the input is a string or list of strings, they will be converted to integers before comparison.

AutoTestLib.api.request_se(structural_element)¶

this function will read the content of the structural element file and return it as a string. If the structural element does not exist, it will log an error message and return an empty string.

Parameters:

structural_element – Name of the structural element (with or without .cm extension)

Returns:

Content of the structural element file as a string, or an empty string if the file does not exist.

Examples

>>> request_se("my_structural_element")
'Content of the structural element file'
