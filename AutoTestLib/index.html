AutoTestLib Documentation¶

AutoTestLib¶

Introduction¶

AutoTestLib is a library that is created mainly for operate on test data, and interfacing with all the instruments in the EMC and Smart Building Laboratory in Vittuone, Milan, Italy.

Current Supported Instruments¶

  • ABB

    • M4M and D11-13 power metre product line

      • Modbus RTU

      • Modbus TCP

  • Zera generators

    • COM3003

    • MT3000

    • MT786

  • Keysight

    • M3458A
